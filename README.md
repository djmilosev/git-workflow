# infra-git-flow-v2

## Testing: 


### 1) Sandbox

    1.1) Create a feature branch from sandbox,
    1.2) Make the changes on a feature-sandbox branch,
    1.3) Create a pull request against sandbox,
    1.4) Merge the changes to sandbox.

### 2) Dev

    2.1) Create a feature branch from dev,
    2.2) Create the pull request to merge feature-sandbox to feature-dev,
    2.3) Create a pull request from feature-dev against dev,
    2.4) Merge the changes to dev.

### 3) Master

    3.1) Create a feature branch from master,
    3.2) Create the pull request to merge feature-dev to feature-master,
    3.3) Create a pull request from feature-master against master,
    3.4) Merge the changes to dev.
